Інформаційна система

Інформаційна система "Комп'ютерні комплектуючі" призначена для забезпечення інформацією для користувачів які цікавляться комп'ютерною технікою. Можливість пушуку за типом комплектуючих.

  • Операти́вна па́м'ять (англ. Random Access Memory, дослівно — пам'ять з довільним доступом, первинна пам'ять) — пам'ять ЕОМ, призначена для зберігання коду та даних програм під час їхнього виконання. У сучасних комп'ютерах оперативна пам'ять переважно представлена динамічною пам'яттю з довільним доступом DRAM.

    Перейти

  • Відеока́рта (англ. video card, також графічна карта, графічний адаптер, графічний прискорювач) — пристрій, призначений для обробки, генерації зображень з подальшим їх виведенням на екран периферійного пристрою.

    Перейти

  • Миша (англ. mouse, англ. mouse devices) — один із вказівних пристроїв уведення (англ. pointing device), який дає змогу користувачеві через інтерфейс взаємодіяти з комп'ютером.Іншими різновидами маніпуляторів, що відтворюють рухи руки, є трекбол і графічний планшет.

    Перейти

  • Наву́шники — пристрій для персонального прослуховування музики або інших звукових записів, в комплекті з мікрофоном можуть використовуватись як засіб для ведення переговорів по телефону або іншому засобу голосового зв'язку.

Апаратне забезпечення

Апара́тне забезпе́чення або залі́зо (англ. hardware) — комплекс технічних засобів, який включає електронний пристрій і, зокрема, ЕОМ: зовнішні пристрої, термінали, абонентські пункти тощо, які необхідні для функціонування тієї чи іншої системи; фізична частина ЕОМ..

Перефирійні пристрої

Перифері́йний при́стрій — частина технічного забезпечення, конструктивно відокремлена від головного блоку обчислювальної системи. Периферійні пристрої мають власне керування і функціонують за командами центрального процесора. Периферійні пристрої призначені для зовнішньої обробки даних, що забезпечує їх підготовку, введення, зберігання, керування, захист, вивід та передачу по каналах зв'язку.

Портативні пристрої

флешки, роутери, блоки живлення, адаптори та інше.
